# 10. Component Rules

- components must be reusable and not page-specific unless necessary
- spacing and typography should follow the design system
- avoid duplicated styling logic across many files
- project cards and service cards should share consistent surface language
- every interactive component should have hover/focus states
- all components must be responsive
