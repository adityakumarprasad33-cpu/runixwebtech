# Runix Web Technologies  SEO Guidelines

---

# 1. SEO Goal

The goal of SEO for the Runix Web Technologies  website is to help relevant people discover the brand when searching for services such as:

- website development
- landing page design
- dashboard development
- web app / MVP development
- website redesign
- developer portfolio / product-focused web services

The site does **not** need bloated SEO spam pages. It needs clean, relevant, service-oriented SEO.

---

# 2. Core SEO Principles

- every page must have a unique title and description
- page copy should reflect actual services and work
- headings should be structured properly
- project pages should be indexable if they add value
- avoid keyword stuffing
- prioritize clarity and relevance over gaming search engines

# 7. Heading Structure Rules

- every page should have one clear H1
- H2s should define major sections
- H3s should be used for service cards, feature groups, process items, etc.
- do not skip heading hierarchy carelessly

---

# 8. Content SEO Rules

- homepage should clearly mention what Runix Web Technologies  builds
- services page should describe service types in natural language
- work page should include meaningful project descriptions
- project summaries should explain what the project is, not just list tech
- avoid vague copy with no search relevance

---

# 9. Technical SEO Requirements

- use semantic HTML
- ensure fast load times
- optimize images
- keep layout shift low
- use descriptive alt text for important images
- generate sitemap later
- include robots configuration if needed
- ensure mobile responsiveness

---
